# Robot framework Project

## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/jalknToth/botFrame.git
   ```

2. **Run the setup script:**

   ```bash
   chmod +x run.sh
   ./run.sh
   ```

3. **Input the files**

4. **Run the robot:**

   ```bash
   robot tests/file_analysis.robot
   ```

## Dependencies

- robotframework 
- robotframework-pythonlibcore
- openpyxl
- python-docx
- pypdf2
- pandas
- reportlab
